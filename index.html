Meow

This repository contains several things I've used for Rust testing.

  • Inside src/strings/ there are some older tools, one of which relies on the lolcat bible for benchmarking.
  • At src/testloop.rs there's source for testloop, a tool for TDD in Rust. Running the following gives you constant feedback when you change a *.rs file under the directory of the crate file given, for example, under src/: sh make ./bin/testloop src/demo.rs

I've also got the beginnings of a presentation about this here.
